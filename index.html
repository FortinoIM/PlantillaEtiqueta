HTML

Html: indica el comienzo del documento HTML.

Body: es el cuerpo de la página, donde va lo que se ve en el navegador al cargar una web. En el body van los textos, las imágenes y todos los contenidos de la web.

Title: Es el titulo de nuestra pagína

A: define los enlaces.

P: el texto dentro de esta etiqueta forma un párrafo.

  • Learn More

CSS

Width: Ancho de un elemento.

Margin: Espacio que se añade entre el elemento y sus vecinos. Se puede diferencia por lado (arriba, abajo, izquierda, derecha).

Color: Color del elemento. Se puede especificar en diferentes formatos como palabras predefinidas (red, green, etc.) RGB o como valor hexadecimal.

Background-color: Color del fondo del elemento.

Padding: Relleno interior que se añade en los bordes del A diferencia de margin, cuenta para el tamaño del elemento.

  • Learn More

JavaScript

Var: Las variables en JavaScript se definen usando esta palabra clave

Document.title: Muestra el título de la página actual

Document.getElementById(): Permite identificar un elemento en una página por su identidad (ID).

Location.href: Encadenando location.href se refresca el contenido de la página (vuelve a cargarla).

History.back(): Página anterior en el historial
